# KRP Barbershop

En statisk webbplats för KRP Barbershop byggd med HTML, CSS, JavaScript och Bootstrap.

## Om projektet

Denna webbplats visar en modern och mobilvänlig layout för en barberarsalong med:
- startsida med bokningsformulär
- informationssida om tjänster
- om oss-sida
- kontaktsida
- användning av animeringar, ikoner och responsiv navigation

## Tekniker

- HTML5
- CSS / SCSS
- JavaScript
- Bootstrap 5
- Google Fonts
- Font Awesome
- AOS (Animate On Scroll)

## Projektstruktur

- `index.html` – startsida med booking-widget
- `våraTjänster.html` – lista över tillgängliga barberartjänster
- `omOss.html` – om salongen
- `kontaktaOss.html` – kontaktinformation
- `bokningsBekräftelse.html` – bokningsbekräftelse
- `css/style.css` – färdig CSS
- `scss/style.scss` – SCSS-källa
- `js/KRP-Barbershop.js` – huvudsaklig JavaScript-logik
- `js/email-config.js` – e-postkonfiguration för kontaktformulär
- `Images/` – bildfiler och logotyp
- `package.json` – projektmetadata och beroenden
